
# [Number the Stars Quiz Answers Exporter](https://github.com/Natf/google-apps-script/tree/master/Number-the-Stars)
This was made to easily export the quizzes, which the students completed using google forms, to PDF files. This project is to easily print and give to the kids a clear record of the quiz, their answers, and the correct answers.
